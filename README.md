# Scaling Notes

Short notes and lessons learned from building and scaling organizations.

---

## Key Lessons

- Structure scales faster than people  
- Clarity beats speed when complexity increases  
- Most operational issues are communication issues  
- Predictability is more valuable than peak performance  

---

## Focus Areas

- Organizational structure  
- Decision-making  
- Execution cadence  
- Scaling challenges  

---

## Context

Based on experience building and scaling a €30M / 90 FTE organization with full P&L ownership.
